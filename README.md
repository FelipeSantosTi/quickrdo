## Sobre o QuickRDO

Este projeto, desenvolvido na linguagem PHP utilizando o framework Laravel, tem por objetivo facilitar o envio de Relatórios Diários de Fase (RDF) em tempo hábil para a elaboração do Relatório Diário de Obra (RDO), bem como registrar histórico das atividades realizadas.
